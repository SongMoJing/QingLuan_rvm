## RVM

> 青鸾软件运行虚拟机

### 青鸾软件包

#### 软件[QLS]结构
```
project.qls (使用zip压缩)
   file: Class Map : 类映射表
   folder: src (静态代码文件夹)
       file: CLASS.qlc (主类)
         ...
   folder: res (静态资源文件夹)
   folder: lib (库文件夹)
```

#### 软件[QLP]结构
```
project.qlp (使用zip压缩)
   folder: ql (静态类文件夹)
       file: CLASS.qlc (类)
         ...
   folder: lib (依赖库文件夹)
```
